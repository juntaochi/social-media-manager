# Teaching AI to Sound Less Like AI

## The Story
One of the hardest parts of this project wasn't the orchestrationâ€”it was the tone. I noticed the Writer agent was defaulting to "I'm thrilled to announce" and using too many bullet points. I built a `voice_calibration.md` system that explicitly bans "AI slop" and teaches the model to write like it's texting a dev friend. The difference in authenticity is night and day.

## Deep Dive Findings
- **The "Hard Ban" List**: `docs/agents/writer/references/voice_calibration.md` includes a strict list of banned terms like "game-changing", "delve", and "harness".
- **Structure Shifts**: The system now favors "one sentence hooks" and casual sentence starters like "finally got X working" over formal announcements.
- **Vibe Test**: Added a "Vibe Test" protocol: "Would I actually say this to a friend at a coffee shop?".
- **Impact**: Shifted the narrative from "explaining architecture" to "sharing a story", resulting in significantly more authentic drafts.
