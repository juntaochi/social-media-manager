# Casual Content Examples

Real examples of the tone we're going for. These sound like humans, not content machines.

## Twitter/X Examples

### Launch Example (Good)

```
shipped the thing

been building an automation that turns my github activity into posts

sounds dumb but I kept forgetting to share what I'm working on

now it reads my commits and drafts posts for me to approve

still need to actually approve everything so no robot takeover yet
```

Why it works:
- Starts with action, no context needed
- Self-aware ("sounds dumb but")
- Explains benefit simply
- Ends with personality

### Struggle/Victory Story (Good)

```

thought it was the API. rewrote it twice.

turns out I had the wrong timezone. my server was in UTC, the API expected local time.

```

Why it works:
- Relatable pain
- Shows the journey
- Humble ending
- "classic" = self-aware humor

## LinkedIn Examples

### Launch (Good)

```
quietly shipped something last week

I kept forgetting to post about what I'm building. like, I'd finish a feature, feel good about it, then forget to tell anyone.

so I built a system that watches my github and drafts posts for me. I still approve everything - not letting AI post on my behalf. but it reminds me what I did and gives me a starting point.

if you're bad at sharing your work too, might build this into something others can use. lmk if that's interesting.
```

Why it works:
- Lowercase "quietly shipped" = humble
- Honest about the problem (forgetting)
- Clear about human control
- Ends with genuine ask
- No hashtag spam

## The Vibe We Want

**Sounds like:**
- A developer friend sharing in a group chat
- Someone excited but not trying to sell you anything
- Honest about limitations and struggles
- Inviting you to follow along, not teaching you

**Doesn't sound like:**
- A company announcement
- A technical blog post
- A LinkedIn influencer
- An AI that read too many marketing guides

## Quick Transforms

| If you catch yourself writing... | Replace with... |
|----------------------------------|-----------------|
| "I'm excited to announce" | "shipped" / "it's live" / "finally done" |
| "Here's what I learned" | "turns out" / "honestly didn't expect" |
| "The key takeaway is" | just end naturally |
| "In this thread I'll cover" | just start |
| "Let me break this down" | just break it down |
| "What are your thoughts?" | "anyone else deal with this?" |
| Headers and sections | just paragraphs |
| Perfect bullet points | messy casual lists or none |
